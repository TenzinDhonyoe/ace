# Exam structure

## Course
CS240 — Data Structures & Algorithms

## Format
- ~20 MCQs on complexity + correctness
- Problem-solve: pick the right data structure for 3 scenarios

## Sections to cover
1. Big-O complexity analysis
2. Sorting algorithms (quicksort, mergesort, heapsort)
3. Hash tables (collision resolution, load factor)
4. Graph traversal (BFS, DFS, Dijkstra)
